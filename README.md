Este repositorio contiene el código fuente de una aplicación de gestor de instalaciones desarrollada para la Universidad Complutense de Madrid (UCM). 
La aplicación utiliza tecnologías como EJS, HTML, JavaScript, AJAX, Node.js, Express y una base de datos MySQL gestionada a través de XAMPP. El objetivo de la aplicación es facilitar la gestión eficiente de las instalaciones disponibles en el campus.

Características Principales
Motor de Plantillas EJS: EJS (Embedded JavaScript) se utiliza como motor de plantillas para generar dinámicamente las páginas HTML en el lado del servidor, facilitando la presentación de datos y la creación de vistas interactivas.

Frontend Interactivo con JavaScript y AJAX: La interfaz de usuario se ha mejorado mediante el uso de JavaScript y AJAX para proporcionar una experiencia de usuario más dinámica. La aplicación realiza solicitudes asíncronas al servidor para actualizar contenido sin necesidad de recargar la página.

Servidor Node.js con Express: La lógica del servidor está implementada en Node.js, aprovechando el framework Express para la gestión de rutas y peticiones HTTP. Esto proporciona una estructura organizada y eficiente para la aplicación.

Base de Datos MySQL con XAMPP: La aplicación utiliza una base de datos MySQL para almacenar información relevante sobre las instalaciones en el campus. XAMPP se utiliza como entorno de desarrollo local para gestionar la base de datos.
